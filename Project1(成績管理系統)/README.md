# 第一章專案：學生成績管理系統

## 專案描述

設計並實作一個簡單的學生成績管理系統，此專案將綜合運用本章所學的C語言基礎概念，包括：
- 基本資料型別
- 陣列操作
- 函式設計
- 輸入輸出
- Makefile使用
- 模組化程式設計

## 功能需求

### 核心功能
1. **學生資料輸入**：輸入學生姓名和成績
2. **成績計算**：計算平均成績、最高分、最低分
3. **成績統計**：統計各等級人數（優秀、良好、及格、不及格）
4. **資料顯示**：顯示所有學生資料和統計結果
5. **資料驗證**：確保輸入的成績在有效範圍內（0-100）

### 等級劃分標準
- 優秀：90-100分
- 良好：80-89分
- 及格：60-79分
- 不及格：0-59分

## 技術要求

### 1. 程式結構
- 使用模組化設計，至少分成3個檔案：
  - `main.c`：主程式
  - `student.c`：學生相關函式實作
  - `student.h`：函式宣告和常數定義

### 2. 資料結構
- 使用陣列儲存學生資料
- 最多支援30名學生
- 每個學生包含：姓名（最多20字元）、成績（0-100）

### 3. 函式需求
必須實作以下函式：
```c
// 輸入學生資料
void input_students(char names[][21], int scores[], int *count);


// 計算平均成績
double calculate_average(int scores[], int count);

// 找出最高分
int find_max_score(int scores[], int count);

// 找出最低分
int find_min_score(int scores[], int count);

// 統計各等級人數
void count_grades(int scores[], int count, int grade_count[]);

// 顯示所有學生資料
void display_students(char names[][21], int scores[], int count);

// 顯示統計結果
void display_statistics(int scores[], int count);

// 驗證成績是否有效
int is_valid_score(int score);
```

### 4. 輸入輸出格式
- 友好的使用者介面
- 清楚的提示訊息
- 格式化的輸出結果

## 檔案說明

### 必要檔案
1. `main.c` - 主程式檔案
2. `student.h` - 標頭檔案
3. `student.c` - 函式實作檔案
4. `Makefile` - 編譯規則檔案
5. `test_data.txt` - 測試資料檔案（選擇性）

## 編譯和執行

使用提供的Makefile編譯專案：
```bash
make          # 編譯程式
make run      # 編譯並執行
make clean    # 清理編譯檔案
make test     # 執行測試（如果有）
```

## 提示

1. **開始前先規劃**：畫出程式流程圖
2. **逐步實作**：先實作基本功能，再增加進階功能
3. **測試很重要**：每完成一個函式就測試
4. **注意邊界條件**：如空陣列、無效輸入等
5. **善用Makefile**：頻繁編譯測試

## 擴展挑戰（選擇性）

完成基本需求後，可以嘗試以下擴展功能：

1. **檔案讀寫**：從檔案讀取學生資料，將結果儲存到檔案
2. **排序功能**：按成績或姓名排序顯示
3. **搜尋功能**：根據姓名或成績範圍搜尋學生
4. **更多統計**：標準差計算、中位數等
5. **圖形顯示**：用文字符號繪製成績分佈圖
